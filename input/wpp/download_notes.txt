Several large demography-related data files are required to run the EPI50 pipeline.
We request that the user download these file locally as they are too large for 
github servers. Once downloaded, please move all files to the /input/wpp/ directory. 

Data source: https://population.un.org/wpp/Download/Standard

Files required:
 1) WPP2022_DeathsBySingleAgeSex_Medium_1950-2021.csv
 2) WPP2022_DeathsBySingleAgeSex_Medium_2022-2100.csv
 3) WPP2022_Population1JanuaryBySingleAgeSex_Medium_1950-2021.csv
 4) WPP2022_Population1JanuaryBySingleAgeSex_Medium_2022-2100.csv
